🦇 BatCave

Problema:
A BAT Brasil enfrentava dificuldades na gestão do tempo de resposta aos chamados de manutenção eletrônica. A comunicação manual via telefone impedia a mensuração precisa do atendimento, desde a paralisação da máquina até a solução do problema. Não havia controle eficiente de indicadores como Downtime, MTBF e MTTR.

Solução:
Desenvolvimento de um sistema web para registro, acompanhamento e mensuração automática dos chamados, permitindo controle de tempo de máquina parada e indicadores de performance.

Tecnologias:
HTML • CSS • JavaScript

🌍 World Guessing Game (Ranking Edition)

Conceito:
Jogo educativo que sorteia um país via API e desafia o usuário a adivinhar com base em dicas como capital ou continente. O sistema calcula pontuação dinâmica, aplica bônus matemáticos e salva ranking em banco de dados.

Arquitetura:
✔ POO com separação de responsabilidades
✔ Consumo de API REST
✔ Persistência com SQLite
✔ Interface gráfica com Tkinter

Tecnologias & Conceitos:
